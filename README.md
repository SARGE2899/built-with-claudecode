# Claude Code projects

A collection of learning and experimentation projects.

## Projects

- [`tiny gpt/`](tiny%20gpt/) — a minimal GPT-style transformer built from
  scratch in pure NumPy (no PyTorch, no autograd) to understand how
  transformers actually work internally.
- [`introduction to machine learning/`](introduction%20to%20machine%20learning/) —
  every major ML algorithm implemented side by side on the same tiny dataset,
  organized by category, for direct comparison.

## Shared assets

- [`Readme files/`](Readme%20files/) — shared library of UI/visual assets
  (banners, etc.) used across READMEs in this repo.
