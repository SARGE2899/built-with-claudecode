Tiny GPT

A GPT-style transformer language model built entirely from scratch in pure NumPy — custom attention, a BPE tokenizer, embeddings, a training loop with teacher forcing, and multiple decoding strategies. No PyTorch, no autograd: every forward and backward pass is hand-derived and numerically verified.

Architecture

  • Tokenizer — text ↔ integer ids
  • Token + positional embeddings
  • Causal self-attention — single-head, with causal masking
  • Multi-head attention
  • Layer normalization (forward + backward)
  • Position-wise feedforward blocks
  • Transformer block — attention + FFN + LayerNorm + residuals, stacked
  • Training loop — Adam optimizer, teacher forcing
  • Text generation — autoregressive sampling with temperature
  • Gradient checking — numerical verification of every backward pass

No LoRA, KV-caching, or quantization yet — this build focuses on getting the core transformer math (forward + backward) provably correct by hand.
